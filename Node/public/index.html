Welcome to the REST API

Your go-to API for managing users, expenses, and income seamlessly.

Users Endpoints

  • GET /users - Retrieve all registered users.
  • POST /users - Add a new user to the system.
  • PUT /users/:id - Update an existing user by their ID.
  • DELETE /users/:id - Remove a user from the system.

Expense Endpoints

  • GET /expenses - Retrieve all expense records.
  • POST /expenses - Add a new expense entry.
  • PUT /expenses/:id - Update an existing expense record.
  • DELETE /expenses/:id - Delete an expense record.

Income Endpoints

  • GET /income - Retrieve all recorded income.
  • POST /income - Add a new income entry.
  • PUT /income/:id - Update an existing income record.
  • DELETE /income/:id - Delete an income entry.
